# Cyber Crime in India Dashboard

## Project Overview:
This project presents an interactive Power BI dashboard analyzing cyber crime trends across India. It provides insights into crime distribution, demographics, and regional patterns.

## Features:
- Total arrests and crime statistics
- Gender-wise and age-wise analysis
- State-wise crime distribution (map visualization)
- Crime trends over years
- Interactive filters and visuals

## Tools Used:
- Power BI  
- Power Query  
- Data Visualization
  

## Insights:
- Identified high crime rate states and regions  
- Analyzed gender distribution in cyber crimes  
- Observed trends in crime rates over time  
- Highlighted patterns for better decision-making
